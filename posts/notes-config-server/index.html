Cofiguration server

February 20, 2020 One-minute read
Backend
Server • Backend • ngnix • PM2

View logs ngnix Link to heading 

$ cat /var/log/nginx/error.log
# OR
$ sudo tail -n 20 /var/log/nginx/error.log

Check configuration nginx Link to heading 

$ nginx -c /etc/nginx/nginx.conf -t

Reset service nginx on Ubuntu Link to heading 

$ sudo systemctl restart nginx

Reload configuration of service nginx on Ubuntu Link to heading 

$ sudo systemctl reload nginx

Assign permissitions for user of nginx on Ubuntu Link to heading 

$ chown -R www-data:www-data /var/www/domain.com/subdomain/*

Copy project to folder WWW public Nginx Link to heading 

$ cp -rL /root/projects/project1/dist/public /var/www/domain.me/subdomain/project

Fix permissions for user of nginx on Ubuntu Link to heading 

$ chmod -R 755 */   # or 751 Fix Permissions Folder
$ chmod -R 644 */*  # Fix Permissions Files

Copy project to folder WWW public Nginx and Fix permissions Link to heading 

$ cp -rL /root/projects/Personal/demo_project_backend/app/dist/pwa /var/www/tech-andgar.me/subdomain/pwa && chmod -R 755 */ && chmod -R 644 */*

Link symbolic Link to heading 

$ ln -s /root/projects/Personal/demo_project_backend/app/dist/pwa/* /var/www/tech-andgar.me/subdomain/demo_cobrando

$ ln -sv /root/projects/Personal/demo_project_backend/app/dist/pwa /var/www/tech-andgar.me/subdomain

PM2 Freeze list process for startup Link to heading 

# [PM2] Freeze a process list on reboot via:
$ pm2 save

PM2 remove startup Link to heading 

# [PM2] Remove init script via:
$ pm2 unstartup systemd

Git push Link to heading 

$ git push -u origin master
